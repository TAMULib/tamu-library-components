This component represents a tabs.

A navigation list is a special type of list fine-tuned for being used in navigating a page.

The following may be specified as an attribute on a template element:

  • tab-content: The content to display in a single tab and may have the following attributes:
    • tab-text: The content to display as the name of the tab (on the tab).
